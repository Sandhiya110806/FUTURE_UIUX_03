## 📌 Task Overview
**Lens Flow CRM** is a streamlined Customer Relationship Management (CRM) platform designed to optimize workflow, lead management, and client interaction tracking for photography and visual studios.

### 🌟 Key Features & Focus Areas:
* **Dashboard & Analytics:** Clean visual interface to monitor leads, active projects, and business performance metrics at a glance.
* **Lead & Client Management:** Intuitive user flows for managing prospective clients, tracking inquiry statuses, and scheduling shoots.
* **Interactive Prototype:** Clickable Figma prototype showcasing seamless navigation between client pipelines, project details, and communication tools.
